## Descrição do projeto 

<p align="justify">
  O projeto foi desenvolvido a partir de um curso promovido pela Rocketseat.
  O conteúdo da plataforma poderá ser criado a partir do GraphCMS, que contém toda a estrutura de API necessária para o envio de informações a aplicação e o cadastro de usuários que engressarão no evento.
  
</p>

## Funcionalidades

:heavy_check_mark: Tela inicial para inscição  

:heavy_check_mark: Tela de apresentação dos eventos, vídeos e informações.  

:heavy_check_mark: validação da data de liberação do evento. Caso esteja liberado o evento poderá se acessado e carregará um vídeo no player, caso contrario o evento não estará liberado( Deverá ser ajustado.)
